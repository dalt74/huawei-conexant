Huawei notebook Conexant AC97 coded support scripts

Conexant codec on Huawei/Honor notebooks is "slightly broken".

The scripts included fixes the isuue.

Install:

Just copy files

acpid/* -> /etc/acpid/events
bin/* -> /usr/local/bin

Install and start acpid and enjoy how the sound is working
